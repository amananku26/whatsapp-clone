# whatsapp-clone

[Live Web App](https://whatsapp-clone-53d8c.web.app/)

* ###### Features
  * Users can log in with Google API
  * Can Create Groups
  * Can Switch B/w the groups
  * send messages to any groups
 
* ###### Other Features
  * Last Message Display to the sidebar
  * Last seen on the header of chat
